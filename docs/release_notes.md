# 发布说明

## 公开目的

本仓库公开一条可追溯的 TAAC 2026 方案主线，目标是保留代码、结果和实验背景，便于复现、审阅和后续研究。

本仓库不试图还原所有历史分支，也不发布所有私有尝试。公开范围优先选择结果来源清楚、训练与评测配对明确、设计边界可解释的内容。

## 结果口径

文档中的分数只采用能够较清楚对应到公开榜和历史实验记录的结果。

如果某些后期尝试的来源、命名、提交上下文或训练配置无法清楚确认，则不作为正式主线结果列入。

结果整理遵循以下原则：

- 分数来源可说明；
- train/eval 配对关系清楚；
- 实验变量能够解释；
- 负结果也保留必要背景。

## 公开范围

本仓库包含：

- 一组配对的训练与评测代码；
- 主线模型结构说明；
- 主线实验演化总结；
- 更广泛的实验目录；
- 代表性负结果；
- 参考资料。

本仓库不包含：

- 全量历史实验分支；
- 已训练 checkpoint；
- 比赛平台账号信息；
- 平台缓存和中间产物；
- 仅用于短期试探的临时脚本。

## 复现边界

比赛结果会受到多种因素影响，包括：

- 数据划分和验证方式；
- 随机种子与数据加载顺序；
- checkpoint 选择；
- GPU、CUDA、PyTorch 和 attention backend；
- 公开榜在比赛期间的整体变化。

因此，本仓库更适合作为方案结构、工程组织和实验边界参考，不应被理解为对任意环境下公开榜 AUC 的无条件保证。

## 维护检查

后续维护时建议定期检查以下事项：

1. 检查是否存在私有路径、账号信息、平台缓存或临时产物。
2. 检查第三方文件来源说明是否清楚。
3. 对代码做轻量级 import 或语法检查。
4. 确认 README、模型说明、结果说明和实验目录中的结果口径保持一致。
